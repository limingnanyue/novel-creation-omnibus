# 第8章：文本优化（去AI味完整流程）

> 适用："去AI味""润色""改文""检查AI味"

---

## 8.1 六大门禁

```
Gate A·禁用词替换：
  情态类→仿佛/好像/犹如/宛若/一丝/一抹/些许/隐约
  动作类→深吸一口气/缓缓/不禁/微微/轻轻/淡淡
  表情类→眼中闪过/嘴角勾起/眉头微皱/眉眼低垂
  心理类→心中一动/心头一震/心下了然/心底泛起/不由得
  判断类→不容置疑/不易察觉/显而易见/毫无疑问
  过渡类→突然/瞬间/不由自主/情不自禁/自然而然
  替换原则：禁用词→具体动作/细节描写

Gate B·句式去套路：
  "XX说道"→用动作替代（60%+无标签）
  "……，带着……"万能状语→独立短句
  "他/她知道……"→用行为展示

Gate C·心理描写外化：
  "他很紧张"→"他的手在抖"
  "她很愤怒"→"她一把掀翻了桌子"

Gate D·节奏打碎：
  打断连续排比句
  长句拆短句，段落长短交错
  偶尔用不完整句

Gate E·对话去腔调：
  加入"嗯""哦""行吧""靠""嘶"
  角色可以答非所问
  删掉解释性对话

Gate F·结尾去升华：
  删总结性语句（"他明白了""这一刻……"）
  用动作/场景收尾（"他把烟掐了，回屋睡觉。"）
```

## 8.2 三刀流（番茄爆文特化）

```
第一刀·斩逻辑链：删因为/所以/为了/试图/由于
第二刀·砍形容词：用生理反应替换（愤怒地→抄起烟灰缸砸过去）
第三刀·拆长句：每段不超过3行，每句不超过15字
```

## 8.3 八类必查问题

```
① 解释腔：删掉替读者解释意义的句子
② 作者报账：让账本/菜单/对白自己报账
③ 镜头脚本语言：恢复人物动作链
④ 先果后因：按时间顺序重排
⑤ 主语断裂：一个句群只服务一个动作中心
⑥ 空泛情绪：换成身体反应/动作延迟/物件误碰
⑦ 模板化AI句式：删"不是X而是Y""这意味着""仿佛/好像连用"
⑧ 段尾金句病：段尾要有功能差异，不每段都落判断
```

## 8.4 AI味分级处理

```
轻度（禁用词≤5处/千字）→ Gate A+B
中度（禁用词6-15处/千字）→ Gate A+B+C+D
重度（禁用词>15处/千字）→ 完整六门禁+三刀流+八类必查
```

## 8.6 模型感知去AI味（新增）

在8.1-8.5通用流程执行后，根据运行的模型追加专化修正：

