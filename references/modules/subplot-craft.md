# 副线编织法 · Subplot Craft

## 适用场景
- 副线设计与管理
- 主副线比例控制
- 副线收束时机规划
- 多人物支线交错

---

## 1. 副线的四大类型

### 1.1 浪漫副线（感情线）
| 子类型 | 主线关联度 | 占副线比例 | 典型走向 |
|:-------|:----------|:----------|:---------|
| 并肩成长型 | 高（一起变强） | 40% | 相遇→误会→合作→坦诚→生死→HE |
| 宿命对头型 | 高（立场对立） | 20% | 敌对→被迫合作→理解→暧昧→悲剧/HE |
| 治愈救赎型 | 中（情感支撑） | 20% | 创伤→相遇→治愈→依赖→突破→HE |
| 日常甜宠型 | 低（放松调剂） | 20% | 直接撒糖→各种巧合→HE |

### 1.2 成长副线
- **技能成长：** 学会某个特定技能/技巧（每卷1条）
- **心魔突破：** 角色的内部障碍（全书1-2条）
- **身份认同：** 我是谁/我在哪/我为什么而战（全书1条）

### 1.3 阴谋副线（暗线）
- 贯穿全书的"幕后黑手"线
- 每10-15章提供1次进展
- 每卷末尾揭露部分真相
- 大结局前全部回收

### 1.4 日常支线
```
功能：解压/调剂/真实感
比例：不超过全部内容的10%
常见形式：
- 配角日常（同门/队友的日常生活）
- 美食线（每章1段美食描写）
- 萌宠线（宠物/灵兽的成长）
```

## 2. 主副线的比例分配

### 2.1 建议比例
| 题材 | 主线 | 浪漫副线 | 阴谋副线 | 日常/其他 |
|:-----|:-----|:---------|:---------|:----------|
| 修仙/升级流 | 70% | 10% | 15% | 5% |
| 言情/宠文 | 50% | 35% | 10% | 5% |
| 悬疑/推理 | 55% | 10% | 30% | 5% |
| 都市/职场 | 50% | 20% | 20% | 10% |
| 群像/多人 | 40% | 15% | 20% | 25% |

### 2.2 副线密度公式
```
每10章 = 主线5-7章 + 副线1-3章 + 日常0-1章
```
- 连续5章无副线进展→副线失去存在感
- 连续3章全都是副线→读者忘记主线
- 副线超过10章无进展→读者认为该线被砍

### 2.3 副线的"优先级"管理
| 优先级 | 特征 | 处理方式 |
|:-------|:-----|:---------|
| 🔴 高 | 直接关联主线/最终BOSS | 每5-8章推进1次 |
| 🟡 中 | 关联人物成长/感情线 | 每10-15章推进1次 |
| 🟢 低 | 日常/萌宠/调味 | 每15-20章出现1次 |

## 3. 副线收束时机

### 3.1 副线的生命周期
```
引入期（1-3章）→ 成长期（5-15章）→ 高潮期（2-3章）→ 收束期（1-2章）
```
| 类型 | 总长度 | 推荐的收束位置 |
|:-----|:-------|:--------------|
| 浪漫副线 | 全书 | 大结局前1卷收束 |
| 成长副线 | 1-2卷 | 该卷末尾收束 |
| 阴谋副线 | 全书/系列 | 大结局收束 |
| 日常支线 | 1-3章 | 随时可收 |

### 3.2 收束规则
- 每卷至少收束1条副线（不能全是开放式结尾）
- 每条副线收束前3-5章要有"预兆"（不能突终止）
- 收束后的角色不要直接消失（留个露脸的机会）
- 感情线不要在第1卷就HE（除非是短篇）

### 3.3 开放式副线的处理
有些副线不需要明确的"结局"：
```
感情线：可以停留在"暧昧"阶段
成长线：可以停留在"潜力"阶段
阴谋线：可以留到下一部
```
但必须给读者一个"暂告一段落"的信号。

## 4. 多人物支线交错

### 4.1 支线人物分类
| 人物角色 | 功能 | 副线类型 | 篇幅 |
|:---------|:-----|:---------|:-----|
| 好友/队友 | 战斗搭档/情感支持 | 成长+日常 | 中等 |
| 导师/前辈 | 提供指引/传承 | 成长 | 短→中 |
| 对手/竞争者 | 制造轻度冲突 | 成长 | 中等 |
| 神秘人 | 提供线索/悬念 | 阴谋 | 长 |
| 小人物 | 展现世界观/提供信息 | 日常 | 短 |

### 4.2 支线轮换策略
```
不要试图同时推进所有支线——分批推进。
第1-10章：A线（主线） + 支线1（浪漫）
第11-20章：A线 + 支线2（阴谋进展）
第21-30章：A线 + 支线1 + 支线3（日常）
第31-40章：A线 + 支线2（收束）+ 支线4（新引入）
```

### 4.3 支线人物的"出场-退场"节奏
| 出场频率 | 角色类型 | 退场方式 |
|:---------|:---------|:---------|
| 每3-5章 | 核心配角 | 伴随主线自然推进 |
| 每10-15章 | 间歇性角色 | 完成阶段性任务后退场 |
| 每卷1次 | 卷级角色 | 卷末完成使命后退场 |
| 全书3-5次 | 神秘角色 | 每次提供新信息 |

## 5. 副线的AI味检测

### 检查清单
- [ ] 每10章主副线比例是否合理？
- [ ] 连续5章以上没有副线进展？
- [ ] 副线是否有独立的"起承转合"？
- [ ] 副线收束前是否有预兆？
- [ ] 是否有超过20章没动静的废线？
- [ ] 每卷至少收束1条副线了吗？
- [ ] 支线人物是否有退场计划？
- [ ] 感情线占主线比例是否合适？
- [ ] 日常支线是否超过内容的10%？
- [ ] 每条副线是否服务于某个"主题"？
