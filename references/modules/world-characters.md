# 第3章：世界观构建

> 适用："搭世界观""世界设定""创造世界"

---

## 3.1 八维度构建法

```
① 世界背景：时代/地理/文明程度
② 核心矛盾：这个世界最大的冲突是什么？
③ 力量体系（如适用）：来源/边界/代价/等级
④ 社会结构：阶层/势力/权力分布
⑤ 日常质感：五感锚点各取1-2个
⑥ 特殊规则：与真实世界不同的核心设定
⑦ 时间锚点：故事发生的时代烙印
⑧ 限制与代价：主角不能做什么？代价是什么？

🚨 优先级：冲突时世界观设定拥有绝对优先权
```

## 3.2 角色四维小传（世界观配套）

```
- 核心身份：一句话定位
- 核心执念：这辈子最想守住/得到什么
- 核心软肋：一碰就溃的防线
- 核心矛盾：外在标签与内在底色的反差
```

## 3.3 版本锁定系统

```
- 每个设定文档带版本号（v1.0/v1.1...）
- 锁定状态：🔓（可修改）/🔒（已锁定）
- 新想法→先暂存→后续评估→融入
```

# 第4章：人物全维度刻画（含群像生态）

> 适用："做人设""设计角色""立人设""反派""群像""OOC"

---

## 4.1 五维骨架法（核心框架）

```
① 核心身份：社会定位，决定行为边界（28岁焊工/19岁女知青/戍边将军）
② 核心执念：所有选择的底层动机（攒钱给妹妹治病/守着约定等人）
③ 核心软肋：一碰就溃的防线（怕别人提出身/见不得旁人受冻）
④ 外在标签：旁人第一眼看到的表象（嘴硬心软/爱笑开朗/严肃刻板）
⑤ 内在底色：标签之下的真实本性（嘴硬的人最心软/爱笑的人最孤独）

🚨 铁律：
  - 执念与软肋一一对应
  - 外在标签与内在底色必有反差
  - 所有细节服务于五维骨架
```

## 4.2 六维细节刻画（选2-3个填充）

```
① 专属身体标记：与身份经历强绑定（焊工虎口旧疤/教书先生指尖粉笔茧）
② 惯性小动作：紧张时摸衣角/撒谎时挠后颈/难过时攥口袋里的糖
③ 语言风格：寡言者短句/嘴硬者反问/读书人文气
④ 固定行为偏好：豆浆多加一勺糖/出门三遍检查门锁
⑤ 隐秘小习惯：只有独处时才会做（糙汉焊工晚上绣手帕）
⑥ 关系互动模式：对不同人有不同态度
```

## 4.3 语言四维拆解法

```
① 用词习惯：职业黑话/阶层用词/地域口音
② 句式长短：寡言者1-5字/话多者15字+/急躁者不完整句
③ 语气后缀：老工人"得了"/书生"罢了"/嘴硬者"……关你什么事？"
④ 专属口头禅：必须有来源，与过往经历绑定。每3-5章出现1次

🚨 自检：遮住"XX说"能区分每句话是谁说的吗？
```

## 4.4 反派设计框架

```
① 核心身份：作恶要有身份支撑
② 核心执念：他自认为的"正义/救赎"
③ 核心软肋：人性残留
④ 外在标签：保护色
⑤ 内在底色：行为的底层逻辑

🚨 铁律：
  - 所有恶行必须有自洽逻辑
  - 与主角有镜像关系（相同起点不同选择）
  - 禁写无动机变态
```

## 4.5 群像法则

```
① 一人一锚点（视觉+行为），绝不重复
② 差异化矩阵：性格/能力/立场/执念四维度避免撞型
③ 出场错峰：开篇3章只出核心主角+1个核心配角
④ 焦点轮转法：一场戏1个主焦点(60%)，其他人1个锚点动作
⑤ 同场人数上限：对话场景≤4人，动作场景≤6人
⑥ 撞型自检：互换位置不影响剧情→必须差异化
```

## 4.6 跨卷弧光四阶段

```
卷一·初始态 → 卷二·动摇态 → 卷三·崩塌/蜕变态 → 卷四·稳态

动摇态6信号（选2-3个交叉呈现）：
① 行为破例  ② 动作频率变化  ③ 语言松动
④ 自我怀疑  ⑤ 微表情泄漏  ⑥ 日常习惯断裂
```

## 4.7 人物退场标准

```
核心主角→1-2章+跨卷持续回响（习惯动作/遗物重复触碰）
核心配角→半章-1章+1-2章回旋刀回收
单元配角→1-2段+留1个细节伏笔（后期轻描淡写回收）
工具配角→1句话带过

🚨 退场后不立刻填补功能位置——空缺越久痛感越深
```

## 4.9 众生相容群像生态引擎

### 三笔活人法（让龙套活过来）
```
第一笔：执念——他最放不下的人/事
第二笔：动作——他独有的身体语言
第三笔：台词——他一开口就让人记住的话（≤15字）

✅ 示例：
  执念：曾经是富商，等女儿回来
  动作：不伸手乞讨，只是抬头看人
  台词：（写在眼睛里）「你也有这一天」

检测三问：
① 遮名看三笔→能认出是他吗？
② 去掉任何一笔→还鲜活吗？
③ 只出场一次→读者会记得他吗？
三问全✅才合格
```

### 群像三层架构
```
🟥 核心层(1-3人)：完整弧光+完整档案+≤5章出场
🟧 对照层(5-10人)：局部弧光+精简档案+≤15章出场
🟩 生态层(不限)：三笔活人+需要时出现

动态调级：生态层交汇点≥2→升级对照层
         对照层弧光完成持续出场→降生态层
```

### 因果生命网
```
每个龙套与主线只有一个交汇点，五选一：
❶ 信息型  ❷ 情绪型  ❸ 镜像型  ❹ 催化型  ❺ 涟漪型

🚨 字数红线：龙套与主角互动≤300字
🚨 出场即退场，默认留白90%
```
