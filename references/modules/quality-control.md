# 第12章：结构设计与实时质检

> 适用："结构设计""质检""错别字""逻辑断裂"

---

## 12.1 结构设计按体量适配

```
短故事(1-5万字)：3-8章，开篇快速入题，章章有钩子
短篇(30-50万字)：1卷或上中下，50-100章，三幕结构
长篇(60-100万字)：3-5卷，每卷独立阶段任务+卷末钩子
超长篇(120万字+)：6卷起，卷内可分篇，每卷独立小闭环
```

## 12.2 实时质检三层

```
第一层·字词级：错别字/漏字/多字/同音错字/形近错字
第二层·句段级：句式重复/词汇复现/"说三缺一"/指代断裂/时序跳跃/场景突变
第三层·篇章级：人名/地名/设定一致性/伏笔登记/章末钩子/节奏疲软
```
