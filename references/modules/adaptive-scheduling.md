# 第15章：自适应调度与动态检测

> 适用："快速档""标准档""深度档""自适应""仪表盘"

---

## 15.1 调度哲学（五把尺子）

```
① 题材需要吗？② 用户习惯吗？③ 写作阶段需要吗？
④ 当前内容触发了吗？⑤ 更优方案可用吗？
```

## 15.2 章节自动分类

```
🟢 过渡章：事件密度≤2个/无重大高潮/Chunk内部 → 最低执行
🟡 普通章：事件密度3-4个/有推进非关键 → 标准执行
🔴 关键章：Chunk收尾/重大反转/高潮战斗/情感爆发/卷末 → 全量执行
```

## 15.3 三档写后链路

```
🟢 快速档（过渡章·2步）：
  Step 1: 状态沉淀（实体提取+伏笔更新+摘要）
  Step 2: 基础格式检查（错别字/标点/纯净性）

🟡 标准档（普通章·4步）：
  Step 1: 状态沉淀  Step 2: 硬性检查（逻辑/设定/连续性）
  Step 3: 质量审查  Step 4: 去AI味

🔴 深度档（关键章·6步）：
  Step 1-4: 标准档全部 + Step 5: 剧情线审计 + Step 6: 深度审稿

用户可手动覆盖：说"快速档/标准档/深度档"立即生效
```

## 15.4 写中6种触发检测

```
大段对话(>40%)→对话诊断
情绪场景(情绪词密度>阈值)→情绪链检查
新角色出场(首次出现)→速查+注册关系
世界规则揭示(设定性描述+新术语)→设定核查
行为涟漪(不可逆选择/影响他人)→后果推演
章节钩子(章末检测)→钩子类型+强度审计
```
