> 适用："起点" "飞卢" "晋江" "平台" "签约" "上架" "商业化" "平台风格"
> 核心理念：不同平台的读者是"不同物种"——在起点写得好不等于在飞卢能活。

## 36.2 起点中文网——正统男频

### 起点读者画像

```
● 男，18-35岁，有消费能力
● 阅读经验丰富——看过100+本网文
● 要求：世界观完整、逻辑自洽、不降智
● 弃书原因TOP3：主角降智 / 世界观崩塌 / 节奏拖
● 付费意愿：高（按章节付费，千字3-5分）
```

### 起点题材偏好

```
┌────────────┬────────────────────────────┬──────────────────┐
│ 热门题材     │ 代表作                        │ 字数（完本）        │
├────────────┼────────────────────────────┼──────────────────┤
│ 玄幻        │ 斗破苍穹、完美世界              │ 300-500万字       │
│ 修仙/仙侠   │ 凡人修仙传、一念永恒             │ 300-800万字       │
│ 都市        │ 全职高手、大王饶命              │ 200-400万字       │
│ 历史/架空   │ 赘婿、宰执天下                  │ 300-500万字       │
│ 科幻/末世   │ 流浪地球、全球高考              │ 150-400万字       │
│ 游戏/竞技   │ 全职高手、超神机械师            │ 200-500万字       │
└────────────┴────────────────────────────┴──────────────────┘

🚨 起点铁律：不管你写什么题材——100章内不要再换一个世界
        起点的读者"认赛道"——你写了玄幻就不要突变成科幻
```

### 起点头三章标准

```
【第一章】
□ 主角出场（1段内让读者"认识"主角）
□ 核心设定初显（世界观/金手指/主线任务）
□ 冲突/悬念出现（至少1个让读者"想知道"的东西）
□ 3000字左右，不要超过3500

【第二章】
□ 金手指/核心能力首次展示
□ 第一个小爽点（打脸/升级/获得）
□ 主要反派或第一个对手出场
□ 章末悬念指向"更大的世界"

【第三章】
□ 第一个支线/副本/任务开启
□ 主角展现"与常人不同"的特质
□ 铺垫中期目标（升级/复仇/变强）

🚨 起点特规：
① 前三章不要写"穿越/重生"过程——第一章直接进入新世界
② 不要用大段设定砸读者——设定融入剧情中
③ 开篇不要写"平凡的一天"——直接在冲突中展开
```

### 起点A级签约标准

```
【签约门槛】
● 3万字内：编辑审核，看题材+开头质量
● 10万字：推荐位PK，追读>300
● 30万字：上架，首日订阅>500=存活线
● 100万字：精品频道（均订>3000）

【数据指标（AI不能用，但可以评估自己的表现）】
追读率：>30%=优秀，>20%=及格，<15%=危险
收藏比：追读/收藏 >20%=健康
章节留存：第1章→第10章流失率>60%=开头或节奏有问题
```

### 起点写作风格要点

```
① 语言：偏文雅但不酸
   可以有一定文学性，但不要"卖弄"
   ✅ "他缓缓睁开眼，入目是一间破旧的木屋。"
   ❌ "睁开眼眸，映入眼帘的是一间古朴而沧桑的木质结构居所。"

② 节奏：中速推进，有张有弛
   连续3章打斗后，要有1章日常/修炼/准备
   连续3章日常后，要有1章高潮
   每10-15章一个大高潮

③ 设定：可以大，但要"用得着"
   设定不是"为了好看"——是为了服务剧情
   每一章出场的设定，必须在本章或下章用上

④ 等级体系：清晰+有期待感
   筑基→金丹→元婴→化神（每一步都明确）
   等级不能太多（5-7级最舒适）
   每升一级，要有一个"展示"（别默默升级）
```

## 36.3 飞卢中文网——脑洞+爽点工厂

### 飞卢读者画像

```
● 男，16-28岁，学生+年轻上班族
● 阅读快、换书快、付费冲动
● 要求：爽、快、新——不要"慢热"，不要"铺垫"
● 弃书原因TOP3：前三章不吸引 / 爽点密度低 / 脑洞不够大
● 付费行为：前十章免费，后面收费——前三章决定读者是否付费
```

### 飞卢题材偏好（跟起点完全不同）

```
┌────────────┬────────────────────────────┬──────────────────┐
│ 热门题材     │ 特点                           │ 章节策略            │
├────────────┼────────────────────────────┼──────────────────┤
│ 同人（最强） │ 火影/海贼/漫威/斗罗→套飞卢公式 │ 每章必有"原角色震撼"│
│ 系统        │ 不需要合理，只需要"量大管饱"      │ 每200字一个系统通知  │
│ 穿越/重生   │ 开局就穿，0铺垫                   │ 第一章穿完，第二章爽 │
│ 脑洞        │ 极致的"如果……会怎样"设定         │ 前3章=设定+一次验证  │
│ 直播/综艺   │ 节奏快、弹幕互动多                │ 每章至少2次弹幕高潮  │
└────────────┴────────────────────────────┴──────────────────┘

🚨 飞卢铁律：飞卢读者不看"文笔"——看"脑洞+爽点"
        一段环境描写=劝退。一句"阳光透过窗户洒进来"都是废话。
```

### 飞卢标题公式

```
飞卢标题 = 整本书最重要的营销工具。

【标题公式】[核心设定] + [最强爽点] + [数量级]
✅ "刚被裁员，系统奖励我一家百亿公司"
✅ "穿越海贼，开局觉醒霸王色霸气"
✅ "重生2015，我靠炒币赚了一个亿"

🚨 飞卢标题不能：
❌ 太文艺（"那年花开月正圆"→没人点）
❌ 太模糊（"重生之我要改变命运"→太虚）
❌ 跟风（别人写"百亿"你也写→没差异化）
```

### 飞卢前10章结构

```
飞卢的核心：前10章决定这本书的命运。

【前10章黄金节奏】
章1：穿越/重生/系统降临（第1句就要说明白）
   + 第一个爽点（打脸/获得能力/装逼成功）
   → 1500-2000字，不要超过2000

章2-3：连续打脸/碾压
   每章2-3个爽点，不要停
   反派不要"强大"——飞卢的反派是"用来踩的"

章4-5：小高潮/揭示更大设定
   敌人变强了？主角更强。
   系统升级？爆装备。

章6-10：循环升级
   每章的模式：→ 新挑战 → 秒杀 → 震撼全场 → 新奖励
   💡 飞卢读者要的不是"过程"——是"主角又赢了"的结果

🚨 飞卢禁忌：
① 前三章不要让主角"受苦"——读者会直接弃书
② 不要写主角的"内心挣扎"——飞卢不需要深度
③ 反派不要有深度——只是主角升级路上的垫脚石
④ 每章必须有一个"可以被截图发朋友圈"的爽点
```

### 飞卢更新要求

```
● 最低日更：6000字（3章×2000字）
● 理想日更：9000字（3章×3000字）
● 冲榜日更：12000+字
● 🚨 超过24小时不更新→读者走20%以上

【飞卢存稿策略】
写飞卢不能存超过3天的稿——因为"风向变得太快"
每天写够字数就发，不需要精修
飞卢读者不看"语法有没有错误"——看"爽不爽"
```

## 36.4 晋江文学城——女频+情感驱动

### 晋江读者画像

```
● 女，16-35岁，学生到职场人
● 阅读：深度+情感投入——会为角色哭
● 要求：人设好、感情真、不降智
● 弃书原因TOP3：人设崩塌 / 感情线尬 / 女主人设不讨喜
● 付费：按章节付费，但更看重"收藏"和"评论"
```

### 晋江题材偏好

```
┌────────────┬────────────────────────────┬──────────────────┐
│ 热门题材     │ 特点                           │ 关键要素            │
├────────────┼────────────────────────────┼──────────────────┤
│ 现代言情    │ 职场/校园/都市/破镜重圆          │ 男主魅力>剧情        │
│ 古代言情    │ 宫斗/宅斗/重生/穿书             │ 女主的生存智慧       │
│ 耽美/纯爱   │ 男男感情线，慢热细腻             │ 人设差>感情推进      │
│ 百合/GL     │ 女性之间的情感                   │ 细腻度>爽度          │
│ 奇幻/玄幻言情│ 修仙/西幻/异能+感情线           │ 世界观+感情双线并行   │
│ 穿书/重生   │ 女主穿进某本书/重生改变命运       │ 虐渣+甜宠交替        │
└────────────┴────────────────────────────┴──────────────────┘

🚨 晋江铁律：所有题材的"底色"都是感情——没有感情线的文在晋江活不了
        即使是修仙文/奇幻文，感情线也占40%以上篇幅
```

### 晋江文案（比正文重要）

```
晋江的文案=书的灵魂。读者先看文案再决定是否点开正文。

【晋江文案标准结构】
① 【一句话梗概】（1句话概括全书核心冲突+感情线）
   例："上辈子她为他而死，这辈子她只想离他远远的。"

② 【核心设定+感情线】（1-2段）
   例："一朝重生，她回到了嫁给他的那一天。
   这一次她不再傻傻付出，只想安安稳稳走完剧情就离开。
   可是——为什么他不按剧本走了？"

③ 【阅读提示】（标签/排雷）
   标签：重生 甜宠 虐渣 穿书
   排雷：1v1 / HE / 慢热 / 双洁 / 不喜勿入

🚨 晋江文案铁律：
① 文案不要超过500字（太长读者不看完）
② 文案不要写"剧透"（把高潮写出来=读者不用看了）
③ 文案要写出"感情线"（不是剧情线）
④ 排雷要诚实——骗读者进来=差评+弃文
```

### 晋江前10章结构

```
【前10章：建立感情线的基础】

章1-3：主角+男主出场，交代核心人设
   不要让女主"完美"（没有缺点=没有代入感）
   男主出场要有"记忆点"（一个动作/一句话/一个场景）
   不要"一见钟情"（晋江读者不吃这个）

章4-6：感情萌芽期
   男主女主的第一次"独处"场景
   至少一次"心动瞬间"（小动作/对视/意外接触）
   出现"误会"（感情推进的发动机）

章7-10：第一次危机+第一次亲密
   感情线第一次冲突（吃醋/误会/地位差距）
   第一次"亲密接触"（拉手/拥抱/互救）
   章末留悬念→读者会追更

🚨 晋江节奏：
● 男主不能在前3章就"深情"（太快=假）
● 女主不能"作"（让读者觉得"她有病"=弃书）
● 感情线要有"起承转合"——不是一路甜到底
● 每10章至少一个"让读者尖叫"的场景（壁咚/守护/表白）
```

### 晋江写作风格要点

```
① 语言：细腻但不啰嗦
   注重"感受"——写"她怎么想"比写"发生了什么"重要
    ✅ "她没有说话，只是低着头，手指绞着衣角。"
    ❌ "她觉得很尴尬，不知道该说什么。"

② 节奏：女频可以慢，但不能"空"
   慢=场景描写+心理描写+对话潜台词丰富
   空=没有推进感情线/剧情线的废话
   💡 检查标准：删掉这一段，感情推进受影响吗？

③ 人设：角色要有"记忆点"
   男主：温柔/霸道/高冷/病娇——选一个并贯彻到底
   女主：成长型/虐渣型/咸鱼型——选一个不摇摆
   配角：闺蜜/情敌/助攻——各有功能

④ 感情线：不要"为虐而虐"
   虐要有"合理的误会"——不是"他不说她不问"
   甜要有"对比"——先虐后甜 > 一路甜
   高潮=感情确认（表白/定情/破镜重圆）
```

### 晋江签约标准

```
【签约门槛】
● 1-3万字：编辑审签（看题材+文案+开头3章）
● 签约后：榜单推荐（新晋→季榜→月榜）
● 10万字：入V上架

【重要指标】
● 收藏：决定能否上榜单
   10万字收藏>3000=及格 / >10000=优秀
● 评论数：决定读者互动
   评论/收藏 >10%=活跃
● 营养液：决定年终榜单排名
```

## 36.5 跨平台适配——同一本书改不同平台

### 起点→飞卢适配

```
① 章节：2500字→1500字（砍掉一半描写）
② 节奏：中速→极快（每章爽点翻倍）
③ 标题：文学范→"杀伤性"标题
   起点："往生录"
   飞卢："刚穿越就被系统砸中，我直接无敌了"
④ 反派：有深度→纯工具人
⑤ 删：环境描写/世界观铺垫/配角背景
⑥ 加：系统提示/弹窗/装逼打脸
```

### 起点→晋江适配（男频转女频）

```
① 主角：男→女（性格/审美/社交方式全换）
② 主线：升级打怪→感情驱动
③ 节奏：打斗场景→对话场景
④ 语言：偏刚→偏柔
⑤ 加：心理描写/恋爱线/闺蜜线
⑥ 删：等级体系/打斗细节（1-2句带过即可）
```

### 飞卢→起点适配

```
① 章节：1500字→3000字（加描写/加铺垫/加配角戏）
② 节奏：极快→中速（3章打斗后加1章日常）
③ 反派：工具人→有深度（反派也是人）
④ 加：世界观/等级体系/配角弧光
⑤ 加：前30章不要"秒杀"——要给反派"人物"
⑥ 删：弹幕/系统提示/过快的节奏
```

### 晋江→起点适配（女频转男频）

```
① 主角：女→男
② 主线：感情→升级
③ 节奏：情感推进→事件推进
④ 语言：细腻→直接
⑤ 加：战斗/等级/装备
⑥ 删：大量心理描写（1-2句就够了）
```

## 36.6 平台选择决策树

```
你要写的书→
├─ 男频 → 节奏快? → 飞卢
│       └─ 节奏中/长篇 → 起点
│
├─ 女频 → 感情线为主 → 晋江
│       └─ 快节奏爽文 → 飞卢（女频版）
│
├─ 同人 → 飞卢（绝对优势）
│
├─ 系统/脑洞/直播 → 飞卢
│
├─ 传统玄幻/修仙/历史 → 起点
│
├─ 现代言情/纯爱/耽美/百合 → 晋江
│
├─ 规则怪谈/悬疑 → 起点（偏男）/晋江（偏感情）
│
└─ 短线快文（10万字以内）→ 飞卢或短篇
```

## 36.7 平台文AI味自查

```
【起点AI味自查】
□ 设定是不是"太完美"了？（完美世界观=AI味）
□ 角色是不是"太套路"了？（冷漠师兄/娇蛮师妹 → 加变化）
□ 语言是不是"太工整"了？（每段句式都一样→打破）
□ 等级升级是不是"太公式"了？（每章固定升级→加障碍）

【飞卢AI味自查】
□ 脑洞是不是"跟风"？（别人写过类似的了→换方向）
□ 爽点是不是"太重复"？（每章都打一样的脸→换花样）
□ 标题是不是"太长"？（超过25字→读者记不住）
□ 节奏是不是"太像AI生产"？（千篇一律的结构→加意外）

【晋江AI味自查】
□ 感情线是不是"太急"了？（第一章就深情→太快）
□ 男主是不是"太完美"了？（没有缺点的男主=AI模板）
□ 女主是不是"太被动"了？（永远被男主救→加主动性）
□ 心理描写是不是"太虚"了？（"她的心乱如麻"→具体写怎么乱）
```
