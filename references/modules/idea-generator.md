# 第38章：灵感生成器

> 适用："灵感" "没灵感" "卡文" "想不到" "脑洞" "写作瓶颈" "创意"
> 核心理念：灵感不是"等来的"——是"用公式生成出来的"。

---

## 38.1 核心灵感公式

### 灵感 = 已知元素 × 重新组合 × 约束条件

```
【万能灵感公式】
[元素A] + [元素B] + [反差/跨界/颠覆] = 新灵感

示例：
"古代皇帝" + "直播带货" = 皇帝穿越到现代做直播
"外卖员" + "剑仙" = 外卖小哥是隐退的剑仙
"数学系女生" + "宫斗" = 用数学模型在后宫生存
"海底捞" + "修仙" = 在修仙界开海底捞

💡 每3个元素=1个灵感
   你需要的不是"一个绝妙的点子"——是"大量'还行的点子'"
   从10个"还行的点子"里，一定能挑出1个好的
```

### 灵感组合三轴

```
【轴A：人物身份】
皇帝/杀手/学生/医生/程序员/主播/外卖员/设计师/律师/厨师
【轴B：世界设定】
修仙/末世/宫斗/现代都市/西幻/星际/民国/游戏/校园/娱乐圈
【轴C：核心冲突】
复仇/逆袭/恋爱/生存/解密/争霸/传承/逃离/拯救/成长

灵感 = 轴A × 轴B × 轴C = 1000种组合

示例：
程序猿(A) + 修仙(B) + 逆袭(C)
→ "被裁员后，我写的代码成了修仙功法"
```

## 38.2 八种灵感生成法

```
① 跨界法——把两个不相关的东西放一起
   公式：[领域A的规则] + [领域B的场景]
   例："急诊室规则" + "天庭"
   → "我在天庭急诊科值班的第一天"

② 角色反转法——把常规角色翻过来
   公式：正常职业 + 隐藏身份
   例：清洁工 + 武林第一高手
   → "我是公司保洁，没人知道我是前杀手"

③ 如果……会怎样法——极端假设
   公式：如果[不可能的事]发生了
   例：如果钱突然没用了 / 如果你能听见所有人的内心
   → "突然有一天，所有人的银行卡余额都变成了0"

④ 规则破坏法——打破现有世界的规则
   公式：在[正常世界]中，[某条规则]被打破了
   例：在法治社会中，杀人合法一天
   → "每年有一天，杀人合法。今天就是那天。"

⑤ 时间错位法——把现在和过去/未来混在一起
   公式：[时代A的人]进入了[时代B]
   例：唐朝诗人进入现代诗坛
   → "我靠背唐诗在诗坛封神"

⑥ 视角转换法——从另一个角度看事件
   公式：[常见事件] × [不常见的叙述者]
   例：宫斗×太监视角 / 末世×丧尸视角
   → "我是一只丧尸，隔壁丧尸老哥总给我抢脑子"

⑦ 极限放缩法——把一件事放大或缩小到极端
   公式：[正常事物] × [极端化]
   例：有一个人能记住所有事 / 有一个人忘掉所有事
   → "我能记住每一个投胎前任"

⑧ 规则推演法——把一个设定推到极限
   公式：[初始设定] → 推演 → [极端后果]
   例：如果能无限重生 → 但每次重生世界都会变一点
   → "我重生了100次，但这次世界跟之前都不一样"
```

## 38.3 人物灵感生成

### 人物公式

```
【人物=核心特质+反差+目标+秘密】

核心特质：选择一个（善良/冷酷/毒舌/蠢萌/腹黑/固执）
反差：选择一个（表面做什么 vs 真实是什么）
目标：我想要什么（复仇/保护/自由/认可）
秘密：如果有，一个不能让别人知道的事

例：
"表面上冷漠无情的杀手 + 实际上偷偷养猫 + 想要退出江湖 + 不能让人知道他的软肋"
```

### 人物灵感速查表

```
┌────────────┬──────────────────────────────┐
│ 角色类型     │ 生成公式                        │
├────────────┼──────────────────────────────┤
│ 重生者      │ 前世失败 + 带记忆重生 + 抓住机会  │
│ 穿越者      │ 现代技能 + 古代困境 + 降维打击   │
│ 隐藏大佬    │ 表面平凡 + 背后通天身份 + 被迫出手 │
│ 黑化者      │ 曾经善良 + 遭遇背叛 + 选择堕落    │
│ 守护者      │ 强大 + 有软肋 + 为保护而行动      │
│ 疯子/怪人   │ 独特的逻辑 + 非常规行为 + 背后的理由 │

```

## 38.4 情节灵感生成

### 情节公式

```
【一个好情节 = 角色 × 目标 × 障碍 × 意外】

① 角色想要什么？→ 目标
② 谁在阻止？→ 反派/障碍
③ 发生了什么意料之外的事？→ 转折
④ 角色因此改变了什么？→ 成长
```

```
1. 两个敌对阵营的人被迫合作逃命
2. 主角在人群中发现了"已经死去"的人
3. 一个谎言越滚越大，最后无法收场
4. 主角获得的能力有"代价"
5. 最信任的人其实是叛徒
6. 某个强大存在一直在暗中保护主角
7. 反派的目标和主角一样
8. 主角发现自己不是"被选中的人"
9. 规则被颠覆——好人和坏人标签对调
10. 一个"最不可能"的人救了主角
11. 主角的"外挂"有一天突然消失了
12. 穿越者发现还有人也是穿越来的
13. 主角必须在两个人之间选一个救
14. 胜利的代价比失败更大
15. 主角发现了世界的"真相"（虚拟/循环/被操控）
16. 敌人的敌人不是朋友
17. 主角做了一件"正确但被所有人反对"的事
18. 曾经的朋友变成了敌人
19. 主角获得了预知未来的能力——但每次改变都会带来更坏的后果
20. 一个陌生人对主角说了"奇怪的话"——后来证明是预言
21. 主角为了救某人而牺牲了最重要的东西
22. 两个实力相当的人相遇——不打不相识
23. 主角的"弱点"其实是最大的"武器"
24. 主角被迫与曾经的敌人合作
25. 所有人的记忆都被篡改了——只有主角记得真实
26. 主角发现自己的故事只是一本书/一个游戏
27. 一个"小事"引发了连锁反应——蝴蝶效应
28. 主角在最后关头改变主意——不做"应该做的事"
29. 主角发现最关心自己的人其实是最大的威胁
```

## 38.5 世界观灵感生成

### 世界观公式

```
【世界观 = 核心规则 + 历史 + 冲突 + 细节】

核心规则：这个世界最不一样的一条规则
历史：这条规则是怎么来的
冲突：这条规则造成了什么问题
细节：普通人怎么在这个世界生活
```

### 世界观灵感速成

```
① 反向现实——某条现实规则被反转
   例：在这个世界，说谎的人头上会冒烟 / 考试成绩决定了寿命

② 资源替换——某种常见资源被替换
   例：在这个世界，货币是"时间" / "记忆力" / "声音"

③ 规则叠加——现实规则+一条新规则
   例：现实世界+每个人都有一个系统 / +每个人都能使用魔法

④ 规则删除——现实世界少了一条规则
   例：没有死亡 / 没有重力 / 没有人能说谎

🚨 世界观灵感铁律：
① 一条核心规则就够了——不要堆设定
② 规则必须有"代价"——没有代价的设定=无聊
③ 规则要有"可见的影响"——读者能在前3章感受到
```

## 38.6 灵感到正文的转化

### 灵感评估清单

```
得到一个灵感后，用5个问题评估：

□ 这个灵感能支撑多少字？（1万字短篇？100万字长篇？）
□ 主角是谁？（有没有"人物感"——不只是设定？）
□ 冲突在哪里？（没有冲突的灵感=散文）
□ 第一章怎么写？（300字内能不能吸引人？）
□ 我能不能一口气写3章不停？
  如果前三项都是"是"——这个灵感值得写
  如果最后一项是"否"——这个灵感可能只适合短篇
```

### 灵感管理

```
不要只靠脑子记灵感——用"灵感仓库"管理：

├── 灵感仓库/
│   ├── 长篇候选/        # 能撑100万字以上的灵感
│   ├── 短篇候选/        # 1-3万字能写完的灵感
│   ├── 脑洞碎片/        # 一句话的设定/场景
│   └── 已废弃/          # 当时觉得好但后来否掉的

每个灵感文件格式：
---
标题：[一句话概括]
类型：[长篇/短篇/脑洞]
核心设定：[3句话以内]
第一章开头：[100字以内]
评估日期：[日期]
---

每月扫一次灵感仓库：
→ 把"还觉得好"的排在前面
→ 把"已经不想写了"的移到已废弃
→ 把"脑洞碎片"里成熟的升级到候选
```

---

> **一句话总结**：灵感不是"天降的"——是"生产出来的"。用公式生成大量点子，用清单筛选，用管理工具保存。当别人说"我没灵感"的时候——你已经写好了三个备选方案。
