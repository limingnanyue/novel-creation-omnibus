# 第16章：番茄爆文特化

> 适用："爆文""番茄爆款""标题公式""系统流"

---

## 16.1 平台认知

```
本质：文字版抖音，完读率=金钱
四级流量池：冷启动(点击率)→测试(完读率)→推荐(互动)→爆发
四类用户：18-24要"怪" / 25-35要"爽" / 35-50要"强" / 50-65要"俗"
```

## 16.2 标题四大公式

```
① 疯批逻辑法：[小动作]+[毁灭性后果]
   示例：《我戴上耳塞，楼上全家都死了》
② 偷家战术法：[反派离开]+[毁灭性洗牌]
   示例：《老婆隐居深山，回来家没了》
③ 极致不对等法：[巨大付出]+[侮辱性回报]
   示例：《赚1亿项目，老板给200元》
④ 降维打击法：[通天身份]+[找死挑衅]
   示例：《刚撕了生死簿，院长让我扫厕所》

标题4维度自查：后果(不可逆) × 反差(数量级) × 绝情(断后路) × 地位(碾压)
```

## 16.3 四大导语模板

```
① 生死抉择式："先救她！"+痛感+身份对比
② 荒谬索取式：开局即不要脸的要求+理所当然的态度
③ 极致羞辱式：侮辱性动作+势利眼理由+暗藏底牌
④ 规则怪谈式：反常识场景+【规则一】告示牌+留白恐惧
```

## 16.4 爆文八步法

```
① 主人公（高底牌+低处境的极致反差）
② 目标（死亡倒计时）
③ 障碍（人为恶意，非客观困难）
④ 行动（绝地反击，非内耗）
⑤ 结果（300-500字内有胜负）
⑥ 转折（神反转）
⑦ 高潮（公开处刑）
⑧ 结局（恶有恶报，非圣母）
```

## 16.5 四大脑洞法

```
① 强制关联：极致的土+极致的潮（皇帝+996）
② 量级放大：×1亿倍或×负1亿倍
③ 逻辑颠倒：把惩罚变奖励
④ 系统弹窗：万物数据化
```

## 16.6 番茄系统流爽文特化

```
📏 每章1200-1500字，每200字小爽点，每章≥4个大爽点
🎬 第一章300字内必须出现系统+第一个打脸爽点
📐 段落极短，每段1-2句话，最多不超过3句
🤖 系统提示用【】加粗框起来，单独成段，机械冰冷

8种优先爽点：光速打脸/系统暴击/全场震惊/扮猪吃虎/反杀逆袭/财富暴击/美女倒贴/众人崇拜
4种钩子：爽点预告式/反转式/危机式/奖励式

🚫 禁止流水账/主角被虐超50字/配角抢戏/负能量
```

## 16.7 S/A/B/C数据评级

```
S级：点击率>30% 首章完读>50% 完读率>35%
A级：点击率10-30% 完读40-50% 完读率25-35%
B级：点击率6-10% 完读30-40% 完读率15-25%
C级：点击率<6% 完读<30% 完读率<15%

高点击+低留存→重写前500字
低点击+高留存→改标题
全线崩盘→切书重开
```

## 16.8 写前三阶段自检

```
🟢 写前自检：
□ 目标读者画像？（怪/爽/强/俗）
□ 对应八步法第几步？
□ 冲突是人为恶意吗？
□ 主角有高底牌低处境的极致反差吗？
□ 章末钩子紧扣主线爽点吗？

🟡 写中执行：
① 第1句必须是动作/对话/争吵
② 300-500字一个胜负
③ 阻力来自反派羞辱
④ 主角当场掀桌不忍气吞声
⑤ 章末钩子用动作或对话

🔴 写后复核：
① 删因为/所以/为了/试图
② 愤怒地→抄烟灰缸
③ 段落不超3行
```
