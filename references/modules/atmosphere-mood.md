# 氛围渲染术 · Atmosphere & Mood

## 适用场景
- 情绪氛围构建（压抑/紧张/温暖/诡异）
- 环境渲染与情绪联动
- 群体情绪/大场面氛围
- 悬念氛围制造

---

## 1. 五种基础氛围

### 1.1 压抑氛围
| 要素 | 技巧 | 示例 |
|:-----|:-----|:------|
| 光线 | 昏暗/逆光/阴影拉长 | 夕阳把影子拖得很长，像一道裂隙 |
| 声音 | 低频/单一/重复/间歇 | 空调的嗡鸣、滴水声、远处的叹气 |
| 温度 | 冷/湿/黏 | 空气潮湿得发霉的感觉 |
| 空间 | 逼仄/狭小/无法舒展 | 低矮的天花板、堆满杂物的走廊 |
| 时间 | 缓慢/停滞感 | 秒针每一下都像敲在胸口 |

### 1.2 紧张氛围
| 要素 | 技巧 | 示例 |
|:-----|:-----|:------|
| 节奏 | 加快+短句 | 一步。两步。呼吸屏住。 |
| 感官 | 聚焦到一个点 | 她只盯着那扇门，世界只剩下门把手转动的声响 |
| 心理 | 未知>已知 | 他知道有东西在暗处，但不知道是什么 |
| 环境 | 反常的安静 | 连虫鸣都消失了 |
| 标志 | 不祥的预兆 | 乌鸦飞过、镜子碎了一道痕 |

### 1.3 温暖氛围
| 要素 | 技巧 | 示例 |
|:-----|:-----|:------|
| 光线 | 暖色/柔和/模糊边缘 | 烛光在她脸上镀了一层蜜色的光 |
| 声音 | 舒缓/有规律/生活音 | 水烧开的声音、翻书声、窗外的雨声 |
| 触感 | 柔软/温暖/包裹感 | 厚毛毯裹住全身，脚趾碰到热水袋 |
| 嗅觉 | 食物香/清新 | 烤面包的麦香、洗过的床单味 |
| 动作 | 慢/轻柔/重复 | 她慢慢搅动杯子里的牛奶 |

### 1.4 诡异氛围
| 要素 | 技巧 | 示例 |
|:-----|:-----|:------|
| 违和感 | 一切正常中的不正常 | 房间很干净，但所有相框都朝下放着 |
| 视觉 | 镜像/扭曲/模糊 | 镜子里的她嘴角上翘，而她明明没有笑 |
| 触觉 | 温度异常/无风而动 | 脖子后突然一阵凉意，明明窗户是关着的 |
| 声音 | 来源不明/不该有的声音 | 楼上传来弹珠声——这是顶楼 |
| 逻辑 | 因果割裂/矛盾 | 白天走5分钟的路，晚上走了半小时还没到 |

### 1.5 悲伤氛围
| 要素 | 技巧 | 示例 |
|:-----|:-----|:------|
| 颜色 | 褪色/灰白/冷调 | 世界像褪了色的旧照片 |
| 感官 | 迟钝/麻木 | 咖啡不苦了，烟也不呛了 |
| 物体 | 遗物/空位/未完成 | 桌上摆着两副碗筷，但只有一个人坐下 |
| 动作 | 机械/重复/无意识 | 她一遍遍擦着同一个杯子，直到它发亮 |
| 外界 | 与情绪形成对比或共鸣 | 窗外下着雨（共鸣）/ 窗外阳光明媚（反衬） |

## 2. 氛围的构建节奏

### 2.1 氛围构建三阶段
```
引入（20%）→ 加深（60%）→ 顶点释放（20%）
```
**示例（紧张氛围）：**
1. 引入（20%）：角色意识到环境有异样——"走廊比平时暗了"
2. 加深（60%）：细节不断堆积——脚步声回响、门开了一条缝、某个东西被碰到了
3. 释放（20%）：紧张来源出现或消失——门突然打开 / 原来是猫

### 2.2 氛围的"三叠法"
同一氛围的三种表现，层层递进：
```
压抑1：窗外天色阴沉（视觉）
压抑2：空气闷热得让人喘不过气（身体感受）
压抑3：连平日里吵个不停的蝉都安静了（反常的安静，暗示大过天）
```

### 2.3 氛围反转
氛围忽然切换会产生最大冲击：
```
温暖→诡异：温馨的家庭晚餐中，母亲突然问"你是谁"
紧张→轻松：屏住呼吸冲进房间，发现只是一只猫
悲伤→温暖：在逝者的日记里发现他留下的最后一句话
```

## 3. 氛围与视角的结合

### 3.1 氛围滤网
角色当前的**情绪状态**会影响他们感知到的氛围：
| 角色情绪 | 同一场景的感知差异 |
|:---------|:------------------|
| 开心 | 阳光灿烂、鸟鸣悦耳、行人友善 |
| 难过 | 天灰蒙蒙、声音刺耳、世界冷漠 |
| 焦虑 | 路灯忽明忽暗、脚步声可疑、时间不够 |
| 愤怒 | 颠簸的车辆、嘈杂的人声、每个人都在挑衅 |

**规则：** 氛围必须与POV角色的情绪匹配（除非刻意制造反差）。

### 3.2 多人场景的氛围管理
群像/多人场景中，不同角色的氛围感知可以不同：
```
主角A注意到：房间很温暖（放松状态）
配角B注意到：窗缝漏风（警惕状态）
反派C注意到：角落的阴影比应该有的更深（专业人士的洞察）
```

## 4. 悬念氛围制造

### 4.1 悬念的四层递进
| 层次 | 读者知道 | 效果 |
|:-----|:---------|:-----|
| L1 | 有事要发生 | 好奇（"接下来呢？"） |
| L2 | 知道危险但主角不知道 | 紧张（"别过去啊！"） |
| L3 | 知道部分真相，还不够 | 推理（"原来如此，但还有……"） |
| L4 | 知道一切但无力改变 | 悲剧张力（"不要啊……"） |

### 4.2 "暴风雨前的宁静"
```
线索1：反常的安静（关键氛围信号）
线索2：角色的微小异常（气短/心跳加速）
线索3：环境的微小异常（门开着/东西不在原位）
→ 读者预感：有事要发生
→ 实际事件：可能比读者想的更大或更小（制造"意料之外"）
```

### 4.3 氛围的"留白"技巧
```
写了60%的氛围就够了，留40%给读者脑补。
读者脑补的恐惧/悲伤/温暖，永远比你自己写出来的更强烈。
```
- ❌ "房间里阴森恐怖，鬼气森森……"
- ✅ "房间里什么也没有。但她的呼吸在空气中凝成了一团白雾——六月天。"

## 5. 氛围的AI味检测

### 检查清单
- [ ] 氛围描写是否只停留在视觉层面？
- [ ] 是否有至少2种感官参与氛围构建？
- [ ] 氛围是否与POV角色的情绪匹配？
- [ ] 是否用了"恐怖/可怕/美丽"等直接评判？（删掉）
- [ ] 是否有"三叠法"的层层递进？
- [ ] 氛围的引入→加深→释放是否完整？
- [ ] 有没有使用氛围反转来制造冲击？
- [ ] 是否留了40%给读者脑补？
- [ ] 环境描写是否真的服务于情绪？
- [ ] 同一场景是否有不同角色感知的差异？
