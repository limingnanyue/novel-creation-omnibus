# 第28章：多题材写作模板

> 适用：想写特定题材但不知道写法/想要该题材的套路模板/跨题材尝试
> 核心理念：每个题材有它的"快捷键"——掌握它，而不是从零摸索

---

## 28.1 规则怪谈（Rule Horror / Creepypasta）

> 以"规则清单"为核心驱动力，规则本身制造恐惧

### 核心公式

```
【规则公式】
异常场景 + 【规则X条】 + 规则被打破 + 不可逆后果 + 永远无法离开

【番茄适配公式】
开局30字内出现规则 + 主角不信邪尝试 + 第一次验证（真货） + 后续层叠升级
```

### 写作模版

```
第一章标准结构：
① 日常切入（15秒到位）
   例：「入职第一天，HR递给我一张A4纸，上面的字是红的。」
② 抛出规则（至少3条，逐条验证）
   【员工守则】
   一、电梯只有1、4、7层亮灯。如果在别的楼层停了，不要出去。
   二、地下室没有3楼。如果有人按3楼，不要看他。
   三、夜班不许照镜子。厕所的不行，手机前置也不行。
③ 第一次违规（主角试探/不信邪/被迫违反）
④ 第一次验证（规则=真的）
⑤ 结尾悬钩（更多规则浮现/更恐怖的东西注意到主角）

🚨 规则设计三原则：
① 每条规则在出现时"莫名其妙"→ 后来才恐怖
② 规则之间要有隐藏关联（第2条和第5条说的是一件事）
③ 不能让主角完全遵守规则→ 剧情推进需要违规
```

### 三档风格

```
┌─────────┬────────────────────────────────────────┐
│ 风格     │ 特点                                    │
├─────────┼────────────────────────────────────────┤
│ 硬核恐怖  │ 无解释、无救赎、主角可能死、开放式结局      │
│          │ 适合：起点/偏惊悚悬疑赛道                   │
├─────────┼────────────────────────────────────────┤
│ 暧昧解密  │ 规则背后有真相可挖掘、主角可解谜             │
│          │ 适合：番茄/偏悬疑+爽文赛道                  │
├─────────┼────────────────────────────────────────┤
│ 规则玩梗  │ 规则本身很沙雕/反差（严肃规则+搞笑内容）    │
│          │ 适合：番茄/短篇/抖音衍生                    │
└─────────┴────────────────────────────────────────┘
```

### 规则类型速查

```
【时间规则】"晚12点后不要抬头""凌晨3点整不要看窗外"
【空间规则】"不要进入404房间""别坐靠窗第3排"
【行为规则】"听到敲门声不要应""不要回头看"
【互动规则】"如果有人叫你的全名，别理""不要接座机电话"
【物品规则】"别碰展示柜里的洋娃娃""那张照片不能带回家"
【数字规则】"电梯最多上4个人""数到7必须停"
```

## 28.2 无限流（无限流/Unlimited Flow/快穿）

> 主角穿梭不同副本世界，累积能力/道具，最终超脱

### 核心公式

```
现实主线（10%）+ 副本攻略（70%）+ 成长体系（15%）+ 黑幕揭露（5%）

每个副本=一个完整的小故事单元
副本之间有主线串联（A副本获得的道具=解决B副本的关键）
```

### 副本设计模板

```
【新手副本】（前3个副本，建立规则）
┌────────┬──────────────────────────────┐
│ 要素    │ 示例                            │
├────────┼──────────────────────────────┤
│ 副本名   │ "山村老尸""鬼校""废弃医院"       │
│ 副本规则 │ 生存X天/收集X个物品/找到出口     │
│ 通关条件 │ 解密核心谜题/击败BOSS/完成仪式    │
│ 奖励     │ 技能点/属性强化/道具/队友        │
│ 失败惩罚 │ 扣除寿命/降级/失去道具/队友死亡   │
└────────┴──────────────────────────────┘

【中级副本】（中期，考验团队配合/策略）
┌────────┬──────────────────────────────┐
│ 要素    │ 示例                            │
├────────┼──────────────────────────────┤
│ 副本名   │ "死亡游戏""孤岛大逃杀""时间循环"  │
│ 副本规则 │ 多阵营/有内奸/需要推理           │
│ 通关条件 │ 不一定能全部生还                 │
│ 奖励     │ 稀有技能/关键线索/阵营提升       │
└────────┴──────────────────────────────┘

🚨 无限流铁律：
① 每个副本必须有不同的玩法（不能全是生存模式）
② 主角不一定要在第一副本就最牛逼——但必须有"独特能力"
③ 道具/技能要在后面的副本用得上
④ 副本之间要有"疲惫感"——连续攻略需要休息缓冲
```

### 番茄适配版

```
【字数】每副本3-5万字，每天1-2章
【节奏】300字内进副本，每500字一个小爽点
【系统面板格式】
┌─────────────────────────┐
│ 副本：死亡图书馆         │
│ 难度：★★★★☆             │
│ 规则：夜间不得翻开第7页   │
│ 时间：72小时              │
│ 当前存活人数：8/12        │
└─────────────────────────┘
```

## 28.3 直播文（直播/主播/流媒体）

> 以直播/弹幕为核心互动元素，节奏快、互动感强

### 核心公式

```
直播画面（60%）+ 弹幕互动（20%）+ 现实冲突（15%）+ 系统提示（5%）

关键是：弹幕不是摆设——弹幕推动剧情/提供信息/制造笑点
```

### 弹幕写作技巧

```
【弹幕创作三原则】
① 不要只写"666""牛逼"——弹幕要有不同观点
   "主播你别去啊！" vs "去了才有意思！" vs "已举报"

② 弹幕要有"身份感"——不同的观众说不同的话
   土豪：打赏+简评
   吐槽：专业挑刺
   CP粉：嗑CP
   剧透党：小声剧透（或故意带歪）

③ 弹幕可以"超前知"——但必须是"猜错了的剧透"
   "这个NPC有问题！"（后来发现NPC真的是好人）
   "完了完了完了"（其实没事）
   ［弹幕要利用'读者比角色知道得多'的信息差制造张力］

🚨 弹幕量控制：
  紧张场景：弹幕少/只有几句"别啊"——制造窒息感
  爽点场景：弹幕刷屏——制造参与感
  虐心场景：沉默/只有一两句"……"——最有力
```

### 直播文结构

```
第一章标准模板：
① （直播画面）开局极限操作/搞笑/冲突
   「"兄弟们，今天带你们看点真家伙。"」
② 弹幕出场（2-3条不同观点的弹幕建立氛围）
③ 弹幕里的老粉点破平台规则/现实人设
④ 冲突爆发（对家/榜一大哥/平台规则/现实身份暴露）
⑤ 结尾悬钩（直播间人数暴增/预告下一场）

角色配置：
主角| 要么有特殊技能（游戏/探险/才艺/吃播）
     要么有反差身份（隐藏大佬/落魄富二代/退隐大神）
副线| 现实线（公司/家庭/平台斗争）
     主播间互动（PK/连麦/公会）
     弹幕梗（粉丝团/黑粉/水军/房管）
```

## 28.4 末世/废土（末世/末日/废土/生存）

### 核心公式

```
末日爆发（前3章）+ 生存适应（10章）+ 团队建立（10章）+ 基地建设（10章）+ 势力斗争（∞）

【阶段划分】
┌─────────┬──────────────────────────────┐
│ 阶段     │ 特征                            │
├─────────┼──────────────────────────────┤
│ 混乱期   │ 第1-7天：通信中断/物资匮乏/暴力   │
│          │ 秩序崩溃/只有本能                 │
├─────────┼──────────────────────────────┤
│ 求生期   │ 第7-30天：找到伙伴/基地           │
│          │ 学会用规则（新的）生存             │
├─────────┼──────────────────────────────┤
│ 建设期   │ 第30天+：基地/农业/规则重建        │
│          │ 内部矛盾/管理层腐败/分权斗争        │
├─────────┼──────────────────────────────┤
│ 对外期   │ 与其他幸存者势力相遇               │
│          │ 资源争夺/同盟/背叛                │
└─────────┴──────────────────────────────┘
```

### 末世题材三选一

```
① 传统末世（丧尸/病毒/灾难）
   优势：群众基础好，容易理解
   特点：丧尸设计的差异化=关键（听觉/视觉/进化）
   
② 不可名状末世（克苏鲁/规则污染/维度入侵）
   优势：新，有深度
   特点：恐怖感来自"无法理解"——不需要解释怪物来源
   氛围：「有人疯了，有人成了新的形状，有人在消失。没有人知道明天会发生什么——因为昨天的时间线，已经被抹掉了。」

③ 社会性末世（经济崩溃/秩序瓦解/资源枯竭）
   优势：真实感极强，代入感秒杀魔幻
   特点：没有怪物，人比怪物可怕
   氛围：「一袋大米换一个老婆，三盒烟换一条命，一箱矿泉水换一套房。这就是末日第三个月的物价。」

🚨 末世铁律：
① 不要解释末日的"科学原因"太详细——神秘感比合理性重要
② 不要浪费前3章写"日常"，第一章或第二章必须进入末世
③ 物资的"精准数字"（三包方便面/一瓶半水/28发子弹）比"一点点"真实100倍
④ 不要一个势力撑到底——背叛/内讧/外部压力
```

## 28.5 悬疑推理（悬疑/推理/侦探/反转）

### 核心公式

```
谜题（前1/4）+ 探案（中2/4）+ 反转（后1/4）

每个案件=悬念钩子→线索累积→嫌疑人轮换→第一次反转→真相反转
```

### 案件结构

```
第一章标准模板：
① 案件被发现（尸体/失踪/密室/怪事）
   ——尽量让主角第一个到现场
② 初步判断（表面上的"凶手"）
   ——给读者一个"显而易见"的答案
③ 第一次反转（这个答案不对）
   ——比读者先一步推翻自己的判断
④ 更多线索浮现
   ——新线索指向新嫌疑人
⑤ 章末悬钩（更深的秘密露出冰山一角）

🚨 悬疑铁律：
① 线索必须"公平"——读者有机会和主角同时推理（但不一定推理对）
② 反转必须"意外但合理"——不能让读者觉得"这TM强行"
③ 凶手不能是最后一章才出场的人
④ 不要在解密后"放PPT"解释——用对话/回忆/行动展示真相
⑤ 每1-2章必须有一个新线索/新嫌疑/新反转
```

### 反转设计手册

```
【三重反转标准模板】
第一层反转（中段）：
  "凶手是A" → 不对，A有不在场证明
  💡 技巧：第一层反转要制造"我们之前都错了"的感觉

第二层反转（中后段）：
  "所以凶手是B" → 但B的动机太明显，像是被陷害
  💡 技巧：第二层反转要让读者觉得"这作者有点东西"

第三层反转（结尾）：
  真凶是C —— C可能是"最不可能的人" 或 "全书最不起眼的路人"
  💡 技巧：第三层反转后要在3句话内让读者回想到前面埋的伏笔
        "原来第三页那'一句话'是这个意思！"

【反转辅助工具】
┌──────────┬──────────────────────────────────┐
│ 手法      │ 说明                                │
├──────────┼──────────────────────────────────┤
│ 时间线诡计 │ 利用非线性的时间展示（A先/后/同时）    │
│ 叙事诡计   │ 隐藏叙事者身份/利用视角盲区            │
│ 人物诡计   │ 双胞胎/易容/假死/多重人格              │
│ 信息不对称  │ 不是读者知道的推错了，是信息不全       │
│ 认知陷阱   │ 读者和侦探"以为知道"的是错的          │
└──────────┴──────────────────────────────────┘
```

## 28.6 种田/日常（种田文/日常/田园/慢生活）

> 节奏慢、细节真、人情味重——最考验"真"功力的题材

### 核心公式

```
日常推进（50%）+ 小冲突（30%）+ 人物关系（15%）+ 伏笔（5%）

没有大战、没有生死——但读者就是停不下来
```

### 节奏控制

```
【章节类型配比】
┌────────────┬──────┬────────────────────────────┐
│ 章节类型     │ 占比  │ 内容                          │
├────────────┼──────┼────────────────────────────┤
│ 日常做事     │ 40%  │ 种地/做饭/赶集/串门/带孩子     │
│ 小冲突       │ 25%  │ 抢水源/邻居吵架/买卖纠纷       │
│ 人际关系     │ 20%  │ 相处/互助/相爱/冤家            │
│ 大事件       │ 10%  │ 天灾/生病/孩子上学/老人去世    │
│ 悬念/伏笔    │ 5%   │ 路人说奇怪的话/发现旧东西      │
└────────────┴──────┴────────────────────────────┘

🚨 种田铁律：
① 不要写"今天天气很好，他心情也很好"——用做饭/干农活展示
② 不要只有"种"——种田文="生活文"：种田是载体，生活是核心
③ 日常要有"变化感"——不是每天重复一样的事
   春天→育苗/夏天→防虫/秋天→丰收/冬天→猫冬办年货
④ 小冲突不能只靠"大恶霸"——真实的小冲突是：争地界/邻里闲话/各家孩子打架
```

### 种田文的"真"细节

```
✅ 正确的种田文日常：
"她蹲在菜地里拔草，拔了两行直起腰来，后腰酸得不行。太阳晒得后颈发烫，汗顺着脖子往下淌。她用手背蹭了一下脸，又继续蹲下去。"

❌ 错误的种田文日常：
"她在菜园里辛勤劳作了一上午，虽然很累，但看到长势喜人的蔬菜，心里充满了满足感。"
  （这是AI写的种田文——没有温度感知，没有身体感受，没有具体细节）

【种田文的关键细节】
- 手上的茧子（第几根手指、什么颜色）
- 饭菜的成本（今天这顿饭花了多少钱）
- 时令蔬菜（什么季节吃什么菜）
- 邻里往来的"人情账"（收了人家的礼要怎么还）
- 孩子的学费（具体多少钱，怎么攒的）
- 老人的药费（每天要吃几种药，几块钱）
```

## 28.7 跨题材融合建议

不要只写单一题材——最有竞争力的通常是"题材融合"。

```
┌────────────────┬──────────────────────────────────────┐
│ 融合配方         │ 市场表现                             │
├────────────────┼──────────────────────────────────────┤
│ 规则怪谈+直播文  │ 爆款配方——规则在直播中发现和验证         │
│ 悬疑+无限流      │ 推理需要在副本中完成                    │
│ 末世+种田        │ 末日背景下的田园生活（反差感极强）       │
│ 日常+规则怪谈    │ 日常中逐渐发现异常（细思极恐）          │
│ 直播+无限流      │ 直播攻略副本/直播探险                  │
│ 废土+悬疑        │ 末日后的推理/谁是凶手                  │
│ 种田+系统        │ 田园系统文（种菜得积分/养鸡升级）      │
└────────────────┴──────────────────────────────────────┘

🚨 融合原则：
① 一个题材为主（70%），另一个为辅（30%）
② 不要为了"融合"而强行添加——两个题材要有"化学效果"
③ 融合的"钩子"要提前想清楚（为什么这两个放在一起更好看？）
```

---

> **一句话总结**：题材不是"换皮"——每个题材有它独特的节奏、细节和读者期待。写规则怪谈不能太啰嗦，写种田文不能太快，写无限流不能没有团队。掌握题材的"呼吸感"，比堆砌套路重要一百倍。
