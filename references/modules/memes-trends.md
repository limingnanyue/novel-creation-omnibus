# 第7章：玩梗系统

> 适用："玩梗""加梗""搞笑""沙雕"

---

## 7.1 梗分类

```
① 流行语类："我裂开了""芭比Q了"
② 名场面类：影视动漫经典场景改编
③ 吐槽类：角色内心OS/毒舌评论
④ 反套路类：打破读者预期的梗
⑤ 职业梗：特定职业内部梗
⑥ 时代梗：特定年代集体记忆
```

## 7.2 玩梗技法

```
【自然融入】梗出现要有场景触发，不硬塞
【角色适配】角色说梗要符合人设
【密度控制】每3-5章1-2个梗，搞笑文可加密
【时机把控】紧张场景后→梗缓解情绪；平淡场景→梗制造笑点

🚫 禁用：过时梗(超1年)/烂梗(全网用烂)/硬植梗(与场景无关)
🚫 梗不能超越去AI味红线
```

## 7.3 热梗仓库管理

```
- 每次写章节前检查热梗仓库是否超6小时未更新
- 匹配当前场景融入适合的热梗
- 新热梗自动保存到热梗仓库
```
