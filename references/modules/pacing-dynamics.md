# 节奏动力学 · Pacing Dynamics

## 适用场景
- 快慢节奏交替设计与控制
- 阅读疲劳管理
- 章节长度优化
- 情节密度调整

---

## 1. 节奏基本模型

### 1.1 三种基础节奏
| 节奏 | 特征 | 单章字数 | 阅读感受 | 适用 |
|:-----|:-----|:---------|:---------|:-----|
| 🔴 快 | 短句+高频事件+动作密集 | 1000-1800 | 酣畅/欲罢不能 | 番茄爆文/战斗戏/高潮 |
| 🟡 中 | 叙事为主+适当描写+少量对话 | 1800-2500 | 舒适/推进感 | 日常章节/剧情推进 |
| 🟢 慢 | 大段描写+心理活动+情绪渲染 | 2500-3500 | 沉浸/回味 | 虐心/氛围/关键对话 |

### 1.2 节奏曲线模型
```
事件密度
  ^
  |  🔴快
  |    \        🔴快
  |     🟡中  /  
  |      \   /
  |       🟢慢
  +------------------------→ 情节推进
         ↓读者疲劳点
```
**核心原则：** 读者疲劳前必须加速，否则弃书。

### 1.3 节奏切换信号
| 切换方向 | 触发条件 | 技巧 |
|:---------|:---------|:-----|
| 快→中 | 大战结束/高潮释放后 | 加一段角色反应+环境描写 |
| 中→快 | 推进受阻/新事件触发 | 短句+动作描写+悬念收尾 |
| 慢→中 | 情绪沉淀完/需推进剧情 | 角色的行动决策+新目标 |
| 快→慢 | 信息量过大/需要消化空间 | 角色独白/对话中透露情感 |

## 2. 情节密度控制

### 2.1 四级密度标准
| 密度等级 | 每章事件数 | 每章反转/悬念数 | 读者注意力 |
|:---------|:-----------|:---------------|:-----------|
| 极高 | 5+ | 2-3 | 全程紧绷，不可持续 |
| 高 | 3-4 | 1-2 | 专注，可维持3-5章 |
| 中 | 2-3 | 0-1 | 舒适，可长期维持 |
| 低 | 1-2 | 0 | 松弛，需要事件触发 |

### 2.2 章节密度公式
```
密度 = (事件点数 + 悬念点数 × 2 + 反转点数 × 3) / 章节字数 × 1000
```
- 理想密度区间：0.8-1.5（普通章节），2.0+（高潮章节）
- 连续3章低于0.6→需要加速
- 连续2章高于2.0→需要缓冲章

### 2.3 战斗/对话/描写的节奏比例
| 章节类型 | 战斗:对话:描写:心理 | 阅读速度 |
|:---------|:------------------|:---------|
| 打斗/战斗 | 50%:15%:25%:10% | 快速阅读 |
| 推进剧情 | 10%:30%:25%:35% | 中速 |
| 情感/虐心 | 5%:25%:20%:50% | 慢速沉浸 |
| 日常/铺垫 | 5%:35%:35%:25% | 可变 |

## 3. 读者疲劳管理

### 3.1 疲劳曲线
```
注意力
100% ████████████████
 80% ████████████░░░░
 60% ████████░░░░░░░░
 40% ████░░░░░░░░░░░░ ← 弃书临界点
 20% ██░░░░░░░░░░░░░░
  0% ░░░░░░░░░░░░░░░░
     └─────────────→ 阅读页数
     0  5  10  15  20
```
- 读者注意力从第3章开始下降
- 每5-8章需要一次小高潮重置
- 每15-20章需要一次大高潮重置
- 无高潮=弃书

### 3.2 疲劳重置策略
| 重置类型 | 间隔（章） | 怎么做 | 重置效果 |
|:---------|:----------|:-------|:---------|
| 微重置 | 3-5 | 章末钩子/悬念 | 保持30%注意力 |
| 小重置 | 5-8 | 小高潮/反转/爆点 | 恢复正常注意力 |
| 中重置 | 10-15 | 新人登场/新地图/重大发现 | 重置40%注意力 |
| 大重置 | 15-20 | 卷末高潮/大战/真相揭露 | 重置80%注意力 |

### 3.3 信息释放的"喂食"策略
- **不要一口气喂完：** 不要在一章内把设定/背景/人物关系全抛出去
- **间隔释放：** 每章最多释放2-3个新信息(设定/人物/背景/伏笔)
- **七成原则：** 每次只给读者七成信息，留下三成让他们猜
- **信息前置：** 需要读者记住的信息优先释放，让它们有"发酵"时间

## 4. 章节长度策略

### 4.1 各平台建议章节长度
| 平台 | 建议字数 | 读者阅读时长 | 节奏偏好 |
|:-----|:---------|:------------|:---------|
| 起点 | 2000-2800 | 5-8分钟 | 中→快节奏，信息量中等 |
| 番茄 | 1000-1800 | 3-5分钟 | 快节奏，高密度 |
| 飞卢 | 1500-2200 | 4-6分钟 | 快节奏，爽点密集 |
| 晋江 | 2000-3500 | 5-10分钟 | 中→慢节奏，情感细腻 |

### 4.2 长短章交替策略
```
模式A（爽文）：短→短→中→短→短→短→长（高潮）
模式B（正常）：中→中→短→中→长（高潮）→中→中
模式C（慢热）：长→中→中→短→中→长（高潮）→中
```
- 前3章用短章（建立节奏）
- 每卷最后1章放长章（高潮+收束）
- 闪回/回忆章用短章（信息量小）
- 重大转折用长章（需要空间铺陈）

### 4.3 字数"体检"指标
```
连续5章平均字数 < 平台推荐最小值 → 节奏太快，缺乏细节
连续5章平均字数 > 平台推荐最大值 → 节奏太慢，读者会拖屏
单章字数波动 > ±50% → 节奏不稳定
```

## 5. 各类题材的节奏模板

### 宠文/言情
```
前3章快→相遇/冲突
4-10章中→日常+感情升温
11-15章慢→感情深入/误会
16-18章快→冲突爆发/高潮
19-20章慢→和解/HE
```
### 悬疑/推理
```
前2章快→命案发生/悬念抛出
3-8章中→调查/线索搜集（每章1个新线索+半回收1个旧线索）
9-12章快→关键突破/反转
13-15章慢→真相反转/收束
```
### 爽文/升级流
```
前3章快→金手指/冲突/立目标
4-10章中→升级/打脸（每章1-2次小爽）
11-15章快→强敌/危机
16-18章中→突破/反杀
19-20章快→新地图/新目标
```

## 6. AI味检测·节奏相关

### 检查清单
- [ ] 最近5章是否全是一个节奏？
- [ ] 疲劳点是否已超过5章没有重置？
- [ ] 章节长度是否在平台建议范围内？
- [ ] 每章的事件密度是否在0.8-1.5之间？
- [ ] 3章内是否有充足的节奏变化？
- [ ] 高潮/缓冲章节比例是否5:1左右？
- [ ] 信息释放是否每章不超过3个新信息？
- [ ] 读者是否需要回看才能理解（信息过载）？
