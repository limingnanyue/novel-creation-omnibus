# 描写技法 · Descriptive Craft

## 适用场景
- 动作场面写作（打斗/追逐/对战）
- 环境氛围渲染（都市/自然/建筑/气候）
- 人物外貌与神态描写
- 战斗/战争场景
- 细节具象化（让文字"看得见"）

---

## 1. 动作描写技法

### 1.1 三拍式动作节奏
```
起势（意图）→ 动作执行 → 结果与反应
```

**示例：**
> 他握紧了刀柄（起势）→ 一个箭步冲上前，刀锋斜劈而下（执行）→ 对方向后一仰，险险避过，碎发飘落（结果）

### 1.2 动作描写的四层递进
| 层次 | 描写内容 | 示例 |
|:-----|:---------|:-----|
| L1 表层 | 做了什么 | 他挥拳 |
| L2 细节 | 怎么做 | 他右腿发力，腰身扭转，一记摆拳直奔对方面门 |
| L3 感知 | 身体感觉 | 拳峰传来骨裂的震感，整条手臂发麻 |
| L4 评价 | 效果+心理 | 对方踉跄后退，牙缝渗血——他知道这一拳够了 |

**规则：** 关键战斗至少到 L3，高潮场景到 L4。

### 1.3 快慢交替
- **快节奏：** 短句+动词密集+句号断句（砰。他倒地。血。）
- **慢节奏：** 长句+细节描写+感官放大（那一秒被拉得很长，他能看清飞溅的每一颗血珠）
- **过渡标志：** 关键动作前一秒拉慢，动作中加快

### 1.4 动作+环境的联动
不要孤立写动作，让环境参与：
```
打斗 → 踩碎地板砖/撞翻桌椅/震落墙灰
追逐 → 踢翻摊位/惊飞鸽子/踩进水坑
```

## 2. 环境描写技法

### 2.1 环境五感矩阵
| 感官 | 进阶技巧 | 示例 |
|:-----|:---------|:-----|
| 视觉 | 焦距切换（远景→中景→特写） | 天际线→街道转角→墙上的裂痕 |
| 听觉 | 层次分层（远/中/近/内） | 远方车流→耳边风声→自己的心跳 |
| 嗅觉 | 味道带情绪/回忆 | 雨后泥土味(安全感)、消毒水味(压抑) |
| 触觉 | 质感+温度+湿度 | 潮湿黏腻的空气、生锈的铁栏杆、滚烫的沙砾 |
| 味觉 | 通过口头/呼吸表现情绪 | 满嘴铁腥味(恐惧)、喉咙发苦(绝望) |

### 2.2 环境即情绪
环境描写不是为了证明"文笔好"，而是为了让读者**感受到情绪**。

| 情绪 | 环境特征 | 不宜用 |
|:-----|:---------|:-------|
| 压抑 | 低沉的天空、闷热的空气、逼仄的空间、昏暗的光线 | 明媚阳光、宽敞广场 |
| 温暖 | 柔和的灯光、温暖的香气、柔软的触感 | 冰冷金属、刺骨寒风 |
| 诡异 | 不自然的光源、异常安静、气味矛盾 | 日常温馨场景 |
| 紧张 | 钟表声/滴水声、缩短的视线、影子异常 | 轻松对话、美食描写 |

### 2.3 环境→叙事过渡法
```
环境引入 → 聚焦关键物件 → 物件触发人物反应 → 进入角色视角
```
**示例：**
> 房间很暗，只有窗缝漏进一线光（环境）。光落在桌上的相框上，玻璃反射出一道刺眼的白（聚焦）。他眯起眼，伸手拿起相框（反应）。照片里的人在笑——那是三年前的事（进入叙事）。

## 3. 外貌与神态描写

### 3.1 外貌描写三原则
1. **不堆砌** — 最多3个特征，90%读者记不住更多
2. **有功能** — 外貌特征必须为人物/剧情服务（贵气/杀伐/反差）
3. **留白** — 读者脑补的永远比你写的更美

### 3.2 特征记忆点设计
每个主要角色建议分配**1-2个记忆点**，反复使用：
| 角色类型 | 记忆点示例 |
|:---------|:-----------|
| 主角 | 伤痕/特殊瞳色/标志性配饰 |
| 反派 | 笑容特征/说话方式/固定动作 |
| 配角 | 职业特征/体型/习惯性动作 |

### 3.3 神态×情绪的对照表
| 情绪 | 微表情 | shen体反应 | 声音 |
|:-----|:-------|:-----------|:-----|
| 愤怒 | 咬合肌突出、鼻翼扩张 | 握拳、脖颈紧绷 | 声音压低、语速变慢 |
| 恐惧 | 瞳孔放大(真)、嘴唇发白 | 手心出汗、轻微颤抖 | 声线发紧、断断续续 |
| 悲伤 | 嘴角下拉、眼神空洞 | 肩膀下垂、呼吸变浅 | 沉默的间隔变长 |
| 惊讶 | 眉毛上挑、嘴微张 | 动作停滞 | 短暂失声 |
| 尴尬 | 眼神躲闪、脸颊发红 | 抿嘴、手指绞在一起 | 干笑、调高音量 |

## 4. 细节具象化

### 4.1 抽象→具体的转换公式
```
形容词 + 名词 → 具象化的动作/感官/比喻
```
| 抽象表达 | → | 具象表达 |
|:---------|:--|:---------|
| 他很生气 | | 他攥紧了拳头，指节发白 |
| 她很害怕 | | 她的牙齿在打颤，每次呼吸都像要被呛到 |
| 天很热 | | 柏油路面被晒得反光，空气扭曲得像隔了层水 |
| 他很饿 | | 肚子发出一声闷响，酸水涌到喉咙口 |

### 4.2 细节"三连法"
一个场景至少给三个细节，但不要写满：
```
背景细节（1个）→ 主动作细节（1个）→ 感知细节（1个）
```
**示例：**
> 街角的大排档冒着热气（背景）。老板颠锅的动作熟练得像个杂技演员（主动作）。铁锅里辣椒呛得人嗓子发紧（感知）。

### 4.3 细节的叙事功能
每一个细节必须回答："它为什么要出现在这里？"
- 塑造人物（他桌上有翻旧了的《百年孤独》）
- 暗示情节（花瓶摆在桌沿，随时会碎）
- 制造氛围（墙上时钟的秒针卡住了）
- 伏笔（角落的蜘蛛网上挂着一枚戒指）

## 5. 战斗/冲突场面

### 5.1 三类战斗的写法差异
| 类型 | 重点 | 字数建议 | 节奏 |
|:-----|:-----|:---------|:-----|
| 武术对战 | 招式+攻防转换+距离感 | 300-800字 | 快→慢→快 |
| 枪战/远程 | 掩体+弹道+环境破坏 | 200-500字 | 爆→顿→爆 |
| 群战/战场 | 全景+局部切换+"偶然" | 500-1500字 | 全景→特写→全景 |

### 5.2 战斗的"三秒钟规则"
每次连续动作描写不超过现实中的3秒——超过3秒读者就会失去空间感。
```
正确的做法：3秒动作 → 停顿/反应/对话 → 下一个3秒
错误的做法：连续描写一整段5分钟的打斗
```

### 5.3 战斗后的"身体账本"
打完之后，角色必须"买单"：
- 体力消耗（气喘/手臂发抖/腿软）
- 受伤处理（止血/包扎/疼痛描述）
- 心理影响（应激反应/后怕/亢奋消退）
- 装备损耗（武器损坏/衣服破损）

## 6. AI味检测·描写相关

### 检查清单
- [ ] 是否有"他/她XX地笑了笑"的固定句式？
- [ ] 环境描写是否只有视觉，缺其他四感？
- [ ] 动作描写是否只有"V得Adj"（慢慢地、轻轻地）？
- [ ] 外貌描写是否超过3个特征？
- [ ] 每个细节是否有叙事功能？
- [ ] 战斗场景是否持续超过"3秒"？
- [ ] 抽象描写有没有转化为具象？
- [ ] 神态描写是否用了模板词组（眉头微皱/嘴角上扬）？
- [ ] 环境是否与当前情绪一致？
- [ ] 有没有用"仿佛""似乎""像是"过度修饰？
