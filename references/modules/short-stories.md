# 第5章：短篇创作全流程

> 适用："写短篇""短故事""1万字/2万字/3万字故事"

---

## 5.1 字数结构公式

```
1万字：开15%(1500) + 发55%(5500) + 转15%(1500) + 结15%(1500)
2万字：开10%(2000) + 发40%(8000) + 小转20%(4000) + 高15%(3000) + 结15%(3000)
3万字：开10%(3000) + 发30%(9000) + 一转20%(6000) + 二转15%(4500) + 高10%(3000) + 结15%(4500)

🚨 核心钩子必须在开篇3000字内出现
🚨 终极刀点/反转必须在全文最后5%-10%
🚨 每1000字必须有一个小推进
```

## 5.2 黄金开篇三选一

```
① 日常切入式（最稳妥）：
   [人物]每天[时间]出现在[地点]。[日常动作]。今天也一样。
   不一样的是，[一句反常信息/伏笔，≤20字]。

② 悬念切入式（适配BE/反转）：
   那[物品]在[地点]放了[时间]，[异常状态]，[人物]没[处理方式]。
   没人知道为什么。

③ 对话切入式（适配古风/关系类）：
   "[有潜台词的对话≤15字]"
   "[回应，暴露信息差≤10字]"
   [动作/沉默/细节暗示反差≤30字]

🚫 忌大段背景介绍、忌人物堆出场、忌无目的写景
```

## 5.3 人物三维法

```
① 一个核心标签：一句话定位
② 一个反差点：标签之外的意外特质
③ 一个专属物件/习惯：重复出现的物品或动作

🚨 1万字：核心2人+配角1人
🚨 2万字：核心2-3人+配角1-2人
🚨 3万字：核心3人+配角2人
🚨 核心超过3人→每个人必然立不住
```

## 5.4 冲突三级升级

```
第一级·微澜（发展前期）：日常里的小不对劲
第二级·分歧（发展后期）：第一次正面矛盾，关系出现裂痕
第三级·拐点（转折段）：不可逆事件发生，关系/命运彻底转向

字数位置（1万字）：微澜2500-3500 → 分歧4500-5500 → 拐点5500-7000
```

## 5.5 核心意象贯穿法

```
一个物品贯穿全篇，三次出场：
第一次（开篇·完整鲜活）：建立日常感
第二次（发展·出现裂痕/异常）：暗示关系变化
第三次（结尾·回收全片情感）：刀点/暖点落地
```

## 5.6 结尾七选一

```
① 首尾闭环式：回到开篇场景，状态天差地别
② 留白戛然式：停在未完成瞬间，不写后续
③ 细节反转式：最后一个小细节推翻全文认知（需前文≥3处伏笔）
④ 开放式余味式：不给出明确结局
⑤ 烟火日常式HE：日常场景多了一个人
⑥ 双向奔赴式HE：双向靠近的定格瞬间
⑦ 成长闭环式HE：做出开篇时做不到的事

🚫 禁止结尾升华说教、交代所有后续、空降新信息
```

## 5.7 虐点配置

```
1万字：0-1个轻刺+结尾1个三级穿心刀（刀在9000-9500字）
2万字：1个二级刀(中段)+结尾1个三级刀
3万字：2个二级刀+结尾1个三级刀

高效刀法：首尾闭环刀/意象递进刀/反转钝刀/留白收尾刀
```

## 5.8 HE/温暖向设计

```
① 圆满里藏微小缺口
② 糖点分三层撒：暗示→确认→收束
③ 转折后至少留15%篇幅写和解
④ 冲突越痛，和解越慢越细
```

## 5.9 标题公式

```
悬念式：《那[物品]，[人物]藏了[时间]》
共情式：《[场景/关系]的[时间]，[一件遗憾/温暖的事]》
反差式：《[人设标签]的[地点/物品]，[意外发现]》
意象式：《[短句]，[环境/状态反差]》
```

## 5.10 短篇避坑红榜

```
❌ 堆人物（核心>3个）
❌ 铺设定（花几千字讲世界观）
❌ 开支线（中途加无关剧情）
❌ 结尾升华（"所以说人生就是遗憾"）
❌ 俗套模板（车祸/绝症/雨天死亡/临终告白）
❌ 连续发刀
❌ 信息空降（反转靠新设定圆）
✅ 核心意象重复3次
✅ 对话越短越有力量
✅ 多用名词动词，少用形容词
```
