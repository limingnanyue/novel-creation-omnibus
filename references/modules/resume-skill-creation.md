# 第19章：断更恢复

> 适用："断更重启""好久没写"

---

## 19.1 标准化4步

```
Step 1：快速恢复上下文
  → 输出大纲速览（30秒版+进度+核心人物）
  → 读取记忆台账（确认进度/结尾类型/伏笔状态）
  → 读取对话记忆

Step 2：精简版全检
  → 人设一致性检查
  → 时间线检查
  → 近3章伏笔检查

Step 3：定位当前章节
  → 读取细纲索引确认进度
  → 读取上章细纲 + 当前章细纲

Step 4：就绪提示
  → 「✅ 已就绪，ChX待写，细纲已加载」
```
