## 你什么时候需要它？

- **写网文长篇**：2000-2500字/章，章章有钩子，反AI味六门禁
- **写黄金三章**：300字内入题，800字内打脸，三章留存≥70%
- **写番茄爆文**：标题公式+导语模板+八步法+数据评级
- **写短篇虐文**：字数结构+核心意象贯穿+刀力评分+开放BE
- **写通感痛觉**：六大原型+三段式递进+900行深度素材库
- **去AI味润色**：六大门禁+三刀流+八类必查+分级处理
- **断更重启**：4步恢复上下文，秒回写作状态

## 它会交付什么？

- ✅ **完整的章节正文**（长篇2000-2500字/番茄爽文1200-1500字）
- ✅ **AI味检测报告**（禁用词密度+维度修正对比表）
- ✅ **状态追踪档案**（人物位置/伤势/好感度/伏笔/情绪曲线）
- ✅ **因果链分析报告**（起因→因果链→人物信息差→章节功能）
- ✅ **刀力评分**（5维度×20分，穿心刀/入肉刀/轻刺分级）
- ✅ **通感痛觉描写**（三段式递进：锚定→展开→烙印）

## 它和同类有什么不同？

| 维度 | 其他写作Skill | **本工坊** |
|:-----|:-------------|:-----------|
| 覆盖范围 | 单一写作类型 | 长篇/短篇/爆文/白描/同人全包 |
| 去AI味 | 简单的禁用词替换 | 六门禁+三刀流+八类必查+分级处理+检测报告 |
| 通感痛觉 | 无 | **全行业独一份**：900行通感体系（六大原型+意象库+剂量管控） |
| 质量门控 | 写完了事 | 写→审→改→检→锁稿5阶段闭环 |
| 架构 | 单文件巨无霸 | **模块化路由**：按需加载，token友好 |
| 测试验证 | 无 | 有测试用例和示例产出 |

## 安全边界

- ❌ 不会自动修改你的细纲/大纲/人设文件
- ❌ 不会在不确认的情况下删除任何内容
- ❌ 不会把小说内容写入公开文件
- ✅ 每次写正文前都会读取最新状态文件
- ✅ 所有关键决策（人物命运、剧情转折）等你确认
