# 第9章：衔接修复

> 适用："修衔接""章间跳跃""上下文不通"

---

## 9.1 断裂等级修复

```
轻度（仅章首有时间词）：
  → 删除章首时间词，用上章结尾场景元素替代

中度（一章水了一天无推进）：
  → 保留核心剧情节点，删除所有日常流水账

重度（上下文完全不连贯）：
  → 截取上一章最后一句话作为新章开篇重新续写
```

## 9.2 水文泛滥5步处理

```
Step 1：删除所有"想了想""叹了口气"等废话填充词
Step 2：超过3轮无新信息的对话砍到3轮以内
Step 3：删除所有"没有A，没有B，也没有C"句式
Step 4：删除无信息量的动作描写
Step 5：压缩后至少减少20%-30%字数
```

# 第10章：因果链分析

> 适用："因果分析""逻辑检查""情节合理性"

---

## 10.1 分析流程

```
① 构建因果链：起因→逐级因果→结果
② 人物信息差分析：
   - 人物A知道什么？
   - 人物B知道什么？
   - 读者知道什么？
   - 有什么信息差/误解？
③ 章节功能定位：这章在整本书中起什么作用？
④ 衔接检查：上下章情绪是否连贯？钩子承接是否顺畅？
⑤ 内容成分分析：冲突/日常/信息/伏笔的占比是否合理？
⑥ 输出报告保存
```
